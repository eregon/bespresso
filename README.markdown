# Bespresso

The caffeine logger. A BDD tutorial.

## What you need

* git
* rvm 0.1.38

## Setup

1. rvm install 1.8.7 # only if not present in your RVM rubies
2. rvm use 1.8.7
3. rvm gemset create bespresso
4. git clone git://github.com/jbpros/bespresso.git
5. cd bespresso
6. gem install bundler
7. bundle install
